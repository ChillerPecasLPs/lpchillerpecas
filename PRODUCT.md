## Project
Chiller Peças — landing page de captação de leads B2B para distribuidora de peças para chillers industriais (compressores, válvulas, óleos, filtros, motores, placas).

## Stack
Astro 5, Tailwind CSS v4, Anton (display), Quicksand Variable (sans)

## Brand palette
- Accent: #E86726 (laranja)
- Brand-900: #1A1A1A (quase preto)
- Brand-500: #4A4A4A (cinza médio)
- White: #FFFFFF

## Tone
Industrial, direto, confiável. Não genérico — comunicação de especialista para especialista.
